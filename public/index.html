This tool can help you draw high-quality genomic context graphics.
Please, choose the contexts you want to draw. You can do this with "Draw my contexts" if you know exactly what you want to draw, or you may do a BLAST search with "Search homologous".

This app allows you to generate a genomic context diagram. You can use any genome assembly from GenBank and even use your own .gbff files.


We'll only store the data we need for drawing, and we will delete it afterwards.
